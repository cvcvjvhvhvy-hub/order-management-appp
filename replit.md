# Order Pro - تطبيق إدارة الطلبات

## نظرة عامة
تطبيق ويب تقدمي (PWA) لإدارة الطلبات بين البقالات والتجار. يدعم اللغة العربية ويعمل كتطبيق أندرويد.

## الهيكل
```
├── server.js              # خادم Express الرئيسي
├── public/
│   ├── login.html         # صفحة تسجيل الدخول
│   ├── register.html      # صفحة التسجيل
│   ├── home.html          # الصفحة الرئيسية
│   ├── style.css          # أنماط CSS
│   ├── manifest.json      # ملف PWA
│   ├── sw.js              # Service Worker
│   └── icons/             # أيقونات التطبيق
├── order_management_app/  # كود Flutter الأصلي
└── extracted_app/         # الملفات المستخرجة
```

## الميزات
- **تسجيل الدخول/التسجيل** برقم الهاتف
- **ثلاثة أنواع مستخدمين:**
  - بقالة: إنشاء طلبات ومراجعة العروض
  - تاجر: تقديم عروض أسعار
  - مسؤول: إدارة النظام
- **PWA**: يمكن تثبيته على الأندرويد
- **واجهة عربية** مع دعم RTL
